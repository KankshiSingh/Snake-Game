# Snake-Game
Terminal-based Snake game built in C using the ncurses library. It includes real-time movement, countdown timer for berry, increasing speed with score, pause/resume functionality, high score tracking, and instant restart. Game runs in the Linux terminal and demonstrates real-time input handling, collision detection, and dynamic screen rendering.
